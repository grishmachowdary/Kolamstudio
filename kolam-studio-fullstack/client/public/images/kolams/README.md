## Where to Get Authentic Kolam Images

### 1. Wikimedia Commons (Free, Public Domain)
- Search: "Kolam", "Rangoli", "Muggu", "Pookalam"
- Filter: Creative Commons or Public Domain
- URL: https://commons.wikimedia.org

### 2. Unsplash (Free, High Quality)
- Search: "Rangoli", "Indian floor art"
- URL: https://unsplash.com

### 3. Pexels (Free)
- Search: "Rangoli", "Kolam"
- URL: https://pexels.com

### 4. Flickr (Creative Commons)
- Search with CC license filter
- URL: https://flickr.com

### 5. User Submissions
- Ask users to upload their kolams
- Get permission to use
- Credit the artist

## Image Requirements

- **Format**: JPG or PNG
- **Size**: 800x800px minimum
- **Quality**: High resolution
- **Orientation**: Square (1:1 ratio)
- **View**: Bird's eye view (top-down)
- **Background**: Clean, contrasting floor

## Naming Convention

```
{region}-{occasion}-{number}.jpg

Examples:
- telugu-sankranti-1.jpg
- tamil-margazhi-1.jpg
- kannada-deepavali-1.jpg
```
